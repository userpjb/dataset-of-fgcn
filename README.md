# dataset-of-fgcn
FGCN-VU 数据集介绍
FGCN-VU 数据集是专为四大名著连环画 AI 研究构建的标准化视觉数据集，收录 30 余套四大名著经典连环画，经 “自动筛选 + 人工审核” 得 27929 张有效图像，覆盖工笔重彩、水墨写意等风格及多时期地域版本。围绕 “人脸、人物、画框” 核心实体，以 “模型预标注 + 人工修正” 完成超 15 万个实例标注，建立多层级关联，标注准确率超 91%。经格式统一、色差校正等预处理，图像标准化为 640×640 像素，适配主流 AI 模型，为 GroundingDINO 微调、Stable Diffusion 风格生成等提供数据支撑。
